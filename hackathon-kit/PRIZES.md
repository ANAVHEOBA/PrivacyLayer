# 🎁 Prize Structure & Ideas

Suggested prize tiers and categories for PrivacyLayer hackathons.

## Main Track Prizes

### 🥇 Grand Prize — $5,000
Best overall project combining privacy innovation, technical excellence, and UX.

### 🥈 Runner Up — $2,500
Second-best overall project.

### 🥉 Third Place — $1,000
Third-best overall project.

## Category Prizes ($500 each)

### 🔐 Best Privacy Primitive
Most innovative use of zero-knowledge proofs.
- Novel circuit design
- New privacy guarantees not seen before
- Creative use of nullifiers, commitments, or Merkle trees

### 🎨 Best User Experience  
Most accessible privacy application.
- Non-technical users can understand and use it
- Beautiful, intuitive interface
- Privacy complexity hidden behind good design

### 📚 Best Documentation
Most thorough and helpful documentation.
- Clear architecture diagrams
- Comprehensive threat model
- Tutorial-quality setup guide

### 🔧 Best Developer Tool
Most useful tool for other PrivacyLayer developers.
- SDK extension, CLI tool, or testing framework
- Helps others build privacy apps faster
- Well-documented API

### 🌱 Best Beginner Project
Best project from a first-time ZK developer.
- Shows learning and growth
- Clean implementation of fundamentals
- Good documentation of the learning journey

## Non-Monetary Prizes

- 🏅 **Contributor Badge** — Top 10 projects earn a PrivacyLayer contributor badge
- 🎤 **Demo Day Slot** — Top 3 present at the next community call
- 📝 **Blog Feature** — Winning projects featured on PrivacyLayer blog
- 🤝 **Mentorship** — Winners get 1:1 sessions with core team

## Prize Distribution

- All prizes distributed in **USDC** via Solana
- Winners announced within 48 hours of submission deadline
- Prizes sent within 7 days of announcement
- Disputes handled by a 3-person judge panel (1 core team, 2 community)

## Running Your Own Hackathon

| Day | Activity |
|-----|----------|
| -14 | Announce hackathon, open registration |
| -7  | Release starter kit, host kickoff workshop |
