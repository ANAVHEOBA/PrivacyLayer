# PrivacyLayer Security Best Practices Guide

This guide is organized into three user-friendly sections:

1. [Secure Coding and Operational Hygiene](secure_coding_guidelines.md)
2. [Versioning, Dependencies, and Supply-Chain Safety](secure_versioning_and_dependencies.md)
3. [Code Reviews, Testing, and Release Readiness](code_reviews_and_testing.md)

Recommended reading order:

1. Start with secure coding and note-handling practices.
2. Review dependency and release hygiene before building or deploying.
3. Use the review and testing checklist before opening a PR or shipping a new version.
