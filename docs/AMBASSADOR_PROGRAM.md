# PrivacyLayer Ambassador Program

**Version:** 1.0  
**Launch Date:** March 2026  
**Status:** Active

---

## Program Overview

The PrivacyLayer Ambassador Program rewards community members who help grow our ecosystem through education, content creation, and community engagement.

---

## Program Structure

### Ambassador Tiers

| Tier | Requirements | Benefits |
|------|--------------|----------|
| **Bronze** | 1-500 points | Swag, Discord role |
| **Silver** | 501-2000 points | + Early access, Token rewards |
| **Gold** | 2001-5000 points | + Direct team access, Higher rewards |
| **Diamond** | 5000+ points | + Advisory role, Special events |

### Point System

| Activity | Points |
|----------|--------|
| Create educational content | 50-500 |
| Host community event | 200-1000 |
| Contribute code | 100-1000 |
| Translate documentation | 50-200 |
| Social media engagement | 10-50 |
| Referral (active user) | 100 |
| Bug bounty report | 100-5000 |

## Application Process

### Step 1: Apply

Submit application at: https://privacylayer.io/ambassador

Required information:
- Name and contact
- GitHub username
- Twitter/X handle
- Why you want to be an ambassador
- Relevant experience

### Step 2: Review

- Applications reviewed weekly
- Response within 14 days
- Initial Bronze tier assignment

### Step 3: Onboarding

- Welcome email with resources
- Discord role assignment
- Ambassador kit sent
- First task suggestions

## Rewards

### Token Rewards

Points convert to PLR tokens monthly:

| Tier | Multiplier |
|------|------------|
| Bronze | 1x |
| Silver | 1.5x |
| Gold | 2x |
| Diamond | 3x |

Example: 100 points = 100 PLR (Bronze) or 300 PLR (Diamond)

### Cash Rewards

For exceptional contributions:

| Activity | Reward |
|----------|--------|
| Published article (major outlet) | $100-500 |
| Conference talk | $500-2000 |
| Video tutorial (10k+ views) | $100-1000 |
| Significant code contribution | $500-5000 |

## FAQ

### How long does it take to become an ambassador?

Applications are reviewed weekly. Most decisions within 14 days.

### Can I be an ambassador if I'm not technical?

Yes! We need ambassadors for content, community, translation, and more.

### Do I need to hold PLR tokens?

No, but token holders get bonus points on activities.

### Can I be removed from the program?

Yes, for violating the code of conduct or extended inactivity (3+ months).
