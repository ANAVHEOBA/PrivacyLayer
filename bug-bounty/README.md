# 🐛 PrivacyLayer Bug Bounty Program

> **Protecting Privacy, Securing Funds** — Help us build the safest ZK-proof shielded pool on Stellar.

## Overview

PrivacyLayer runs an ongoing bug bounty program to identify and fix security vulnerabilities before they can be exploited. We reward security researchers with **USDC** for responsible disclosure of valid findings.

### 💰 Reward Pool

**Total Pool**: Up to **$10,000 USDC** per vulnerability  
**Payment Method**: USDC on Stellar Network  
**Platform**: Drips Wave + Immunefi (coming soon)

### 🎯 Scope

| Component | In Scope | Reward Range |
|-----------|----------|--------------|
| **Smart Contracts** | Soroban contracts (Rust) | $500 - $10,000 |
| **ZK Circuits** | Noir circuits | $500 - $10,000 |
| **Cryptographic Ops** | BN254, Poseidon, Merkle trees | $500 - $10,000 |
| **SDK** | TypeScript/JavaScript client | $100 - $5,000 |
| **Frontend** | Web application | $100 - $5,000 |

**Full Scope**: [scope.md](scope.md)  
**Reward Tiers**: [rewards.md](rewards.md)

### 📋 How to Submit

1. **Review** the [scope](scope.md) to ensure your finding is eligible
2. **Prepare** a detailed report (see [submission.md](submission.md))
3. **Submit** to security@privacylayer.io
4. **Wait** for triage (24-72 hours)
5. **Receive** reward upon validation

**Submission Guide**: [submission.md](submission.md)

### ⏰ Timeline

| Stage | Timeframe |
|-------|-----------|
| Initial Response | 24 hours |

| Bounty Decision | 7 days |

### 🚫 Out of Scope

- ❌ Social engineering / phishing
- ❌ DDoS attacks
- ❌ Third-party services
- ❌ Testnet / staging environments
- ❌ UI/UX bugs without security impact
- ❌ Spam / rate limiting issues

### 📜 Rules

1. **Do no harm** — Don't access, modify, or delete others' data
2. **Responsible disclosure** — Report privately first, public after fix
3. **No automated scanning** — Manual testing only (rate limits apply)
4. **One vulnerability per report** — Separate reports for separate issues
5. **No duplicate claims** — First reporter wins
6. **Original work** — Don't plagiarize others' reports

### ✅ Safe Harbor

We pledge:
- ✅ No legal action for good-faith research
- ✅ Confidentiality until fix is deployed
- ✅ Public recognition (if desired)
- ✅ Prompt payment
