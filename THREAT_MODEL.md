# THREAT_MODEL
## Introduction
This document outlines the threat model for the PrivacyLayer project.
## Threat Vectors
### Threat Vector 1: Unauthorized access to the shielded pool
### Threat Vector 2: Zero-knowledge proof manipulation
### Threat Vector 3: Deposit and withdrawal exploits
### Threat Vector 4: Merkle tree manipulation
### Threat Vector 5: Hash function vulnerabilities
## Security Assumptions
### Security Assumption 1: The use of BN254 elliptic curve operations provides sufficient security
### Security Assumption 2: The Poseidon hash function is secure
### Security Assumption 3: The Merkle tree implementation is secure
## Dependencies
### Dependency 1: Stellar Protocol 25
### Dependency 2: Noir
### Dependency 3: Soroban
